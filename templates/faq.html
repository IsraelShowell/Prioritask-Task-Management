Frequently Asked Questions (FAQ)

1. How do I create a new task?

To create a new task, fill out the "Task Name", "Task Description", "Priority", "Task Ranking", and "Task Deadline" fields in the form provided. Once all fields are completed, click the "Add Task" button to save the task.

2. How can I edit an existing task?

To edit an existing task, click the "Edit" link next to the task you want to modify. You will be able to update the task details and save the changes.

3. How do I mark a task as complete?

To mark a task as complete, click the "Mark as Complete" button next to the task. Once completed, the task status will change to "âœ“" and no longer be listed under incomplete tasks.

4. How can I delete a task?

To delete a task, click the "Delete" link next to the task. A confirmation prompt will appear to ensure that you want to delete the task. If confirmed, the task will be removed from your list.

5. How do I search for tasks?

You can search for tasks using the search bar at the top of the page. Enter keywords related to the task name or description and click the "Search" button to filter tasks accordingly.

6. How can I sort the tasks?

You can sort tasks by "Priority", "Task Name", "Task Deadline", or "Date Created" using the sorting dropdown. Select the criteria you want to sort by and choose the order (ascending or descending), then click "Sort" to update the list.

7. How can I filter tasks based on status or priority?

Use the filtering options to display tasks based on their status (completed or incomplete) or priority (all priorities, Important-Urgent, etc.). Choose the appropriate options from the dropdowns and click "Filter" to see the updated list.

8. What is the "Ask Taski" feature?

The "Ask Taski" feature allows you to ask a virtual assistant for help with a task. Click on the "Ask Taski" link next to any task to start the conversation. Taski will assist you with task-related queries.

9. How do I log out?

To log out of your account, click the "Logout" link at the top of the page. This will end your current session and return you to the login screen.
